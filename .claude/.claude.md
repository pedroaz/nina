# App Summary
- Nina is a language learning app
- It is a web application using nextjs
- Choose base language and target language
- Generate Lessons, Exercises, Flash cards all from AI

# Structure
- Monorepo
- packages/core: shared library with most of business logic
- packages/core/cqrs: commands and queries system to update database
- packages/core/llm: ai portion of the system using genkit
- packages/landing-page: nextjs simple landing page app with several premade components
- packages/web-app: nextjs app that interacts with mongodb
